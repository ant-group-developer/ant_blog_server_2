
## Install module packages

```bash
$ npm install
```
## Generate model in Prisma

```bash
$ npx prisma generate
```
## Compile and run the project

```bash
$ npm run start:dev
```

## Check API: 

```bash
http://localhost:3000/api#
```

## .env:
```bash: URL kết nối tới Database
  DATABASE_URL=.....
```

```bash: mã JWT để băm mật khẩu
  JWT_REFRESH_SECRET=.....
```
